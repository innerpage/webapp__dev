## Installation

- Clone the repository <br/>

  ```
    git clone git@github.com:innerpage/webapp__dev.git
    cd webapp__dev
  ```

- Install dependencies <br/>
  ```
    ./init.sh
  ```

<br/>

## Build and Deployment

- Build the code <br/>

  ```
    ./build.sh
  ```
